A great Wordpress plugin to place Ads from AdSense and other Ads anywhere you want.

== Description ==

Please use the new and better Plugin "Easy Code Pleacement". - https://wordpress.org/plugins/easy-code-placement/
This Plugin will not be Supported anymore. "Easy Code Pleacement" is completely rewritten with better handling and a modern Layout.

-----

Easy Ad Placement provides a very easy way to place Ads where you want to have them.

= Features =  
     
* Save up to 10 Ads on options page
* Set the alignement of the Add
* Place a Ad with a Shortcode
* Place a Ad with a PHP-Snippet
* Ready for translations

= Languages =

* Englisch
* German

== Installation ==

= In WordPress =

* Download the Plugin
* Go in your Dashboard to 'Plugins' -> 'Install'
* Click on Upload
* Click on 'Choose file' and choose the downloaded Zip-file
* Click on 'Install'
* Activate the plugin through the 'Plugins' menu in WordPress
* Now you can go to the Settings-Page on 'Settings' -> 'Easy Ad Placement'

= Manually =

* Go in your Dashboard to 'Plugins' -> 'Install'
* Search for 'Easy Ad Placement'
* Click on 'Install'
* Activate the plugin through the 'Plugins' menu in WordPress
* Now you can go to the Settings-Page on 'Settings' -> 'Easy Ad Placement'
